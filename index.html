Curso de Latex y Git


LaTex

LaTex es un potente compositor de textos. Con el mismo contenido, LaTex nos permite generar varios documentos con el simple hecho de cambiar el tipo de documento a generar. En este curso hemos aprendido a generar diferentes documentos en Latex (cartas, artículos, libros,...). Ahora somos capaces de generar complejos documentos con imágenes, listas, enlaces, formulas, bibliografías, ...

Git

Durante el curso hemos aprendido a trabajar con git en local. Ahora somos capaces de gestionar de forma eficiente y eficaz, nuestros proyectos. Git nos permite tener un control de versiones sencillo de iniciar y mantener. Además, somos capaces de crearnos nuestra cuenta github. Este repositorio público, nos permite dar a conocer nuestro trabajo de forma gratuita. A su vez, podemos conocer el trabajo de otros usuarios, colaborar con ellos o viceversa.
